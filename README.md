# Hifz Intel

**Hifz Intel** is a modern, minimal landing page for an online Quran memorization program designed specifically for busy professionals.

The program focuses on smart, structured, and sustainable Quran memorization without requiring participants to sacrifice their careers or personal responsibilities. It combines structured memorization techniques, practical tajwid, guided revision systems, and reflective understanding to ensure long-term retention.

This repository contains a clean, conversion-focused landing page built for clarity, simplicity, and professionalism—making it suitable for platforms such as Carrd, Notion, or any static hosting environment.

## Key Highlights
- Clean and professional UI/UX
- Designed for adult learners and working professionals
- Clear program structure and requirements
- Conversion-oriented section layout
- Easy to customize and deploy

## Tech Stack
- HTML
- Inline CSS (Carrd-ready)
- Responsive layout

## Purpose
This project serves as the official landing page for **Hifz Intel – Smart Quran Memorization for Busy Professionals**, providing a clear overview of the program, its structure, and application flow.
